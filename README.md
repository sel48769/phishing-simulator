# phishing-simulator
This program is designed using JavaFX to create an intuitive and user-friendly application for conducting phishing simulations within an organization. The application generates and sends phishing emails to employees, simulating real-world scenarios to test their ability to identify fradulent emails. It tracks employee interactions, such as clicking on phishing links, and compiles this data for analysis. The program goes beyond just identifying vulnerabilities; it includes an educational component. Employees who fall for the phishing attempts are presented with tailored training modules that explain the dangers of phishing, how to recognize suspicious emails, and best practices for staying safe online. This dual approach of simulation and education helps organizations strengthen their cybersecurity posture by raising awareness and improving employee resilience against phishing attacks.
